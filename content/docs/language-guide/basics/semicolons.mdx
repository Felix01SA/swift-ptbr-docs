# Ponto e vírgula

Ao contrário de muitos outros idiomas, o Swift não exige que você escreva um ponto e vírgula (`;`) após cada instrução em seu código, embora você possa fazê-lo se desejar. No entanto, pontos e vírgula são necessários se você quiser escrever várias instruções separadas em uma única linha:

```swift
let cat = "🐱"; print(cat) // 🐱
```
